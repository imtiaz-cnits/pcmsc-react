Backend Response Handling:

Agar backend se response success: true hota hai:
onSuccess trigger hota hai, jisme success toast dikhaya jata hai.
Agar backend se response success: false hota hai:
onSuccess trigger hota hai, lekin error toast dikhaya jata hai.
Agar API call mein koi error hota hai:
onError trigger hota hai, jisme generic error toast dikhaya jata hai.
